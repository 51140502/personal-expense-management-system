个人收支管理系统
开发语言：C语言
开发工具：Dev-C++、GCC编译器、GitHub
适用场景：个人日常收支记录、消费统计与管理
项目类型：C语言基础实践项目（中北大学大数据与人工智能实验室大一寒假考核）
一、项目介绍
本项目是一款轻量级的个人收支管理系统，基于C语言模块化开发实现，核心解决个人日常收支记录零散、统计不便的问题，支持收支记录的增删改查、多维度统计，并通过文件读写实现数据持久化，关闭程序后数据不会丢失。项目严格遵循Git分批次提交规范，代码结构清晰、注释完善，是C语言基础编程与版本控制工具结合的实践成果。 
二、项目结构
文件名称 功能说明 
 main.c  程序入口文件，实现菜单交互、用户指令接收与核心功能调用 
 record.c / record.h  定义收支记录结构体，实现增删改查、按日期/备注查询、收支统计等核心函数 
 file.c / file.h  实现二进制文件读写功能，完成数据持久化存储与加载 
 Makefile.win  Windows环境下编译脚本，快速编译项目生成可执行文件 
 income_expense.dat  二进制数据文件，持久化存储所有收支记录 
 records.txt  文本数据文件，辅助存储收支记录（兼容文本查看） 
 个人收支管理系统.dev/.layout  Dev-C++项目配置文件，适配开发环境运行 
三、环境配置与运行步骤 
环境要求
- 操作系统：Windows 10/11
- 编译工具：GCC编译器（MinGW）、Dev-C++
- 版本控制：GitHub Desktop
运行步骤
1. 克隆/下载GitHub仓库代码至本地，打开项目目录；
2.（编译器直接运行）：用Dev-C++打开 个人收支管理系统.dev 文件，点击「运行」编译并执行；
3. 运行后根据控制台菜单提示，输入数字指令完成对应操作。
四、核心功能说明 
1. 添加收支记录：输入收入/支出类型、金额、日期、备注，完成记录新增，自动同步至数据文件；
2. 查看所有记录：按序号展示所有收支记录，包含完整信息，清晰呈现收支明细；
3. 按条件查询：支持按日期精准查询、按备注关键词模糊查询，快速定位目标记录；
4. 修改/删除记录：按记录序号进行精准修改或删除，操作后实时更新数据文件；
5. 收支统计：自动计算总收入、总支出、账户余额，同时支持按备注分类统计相关消费金额；
6. 数据持久化：所有操作结果均通过二进制文件保存，程序重启后可加载历史数据，无数据丢失。
五、核心实现要点
1. 模块化设计：将数据结构、业务逻辑、文件操作拆分至不同文件，降低代码耦合度，便于维护；
2. 结构体封装：定义收支记录结构体，统一管理记录的类型、金额、日期、备注等属性；
3. 文件操作：采用二进制文件读写方式，相比文本文件更节省空间、读写效率更高；
4. 边界处理：对用户输入的非法数据（如非数字金额、无效日期）做基础校验，提升程序健壮性；
5. 版本控制：遵循Git规范分批次提交代码（核心源码→数据文件→配置文件），提交记录可追溯，符合实验室考核要求。
